# Office Operations Data Analysis

## Project Overview
This project focuses on cleaning and analyzing office operations data to optimize efficiency, track management metrics, and streamline workflow processes. 

## Tech Stack
* **Data Cleansing:** Excel / SQL
* **Data Source:** Cleaned Office Operations Dataset (`Office_Data_Cleaned.csv`)

## Key Insights & Data Operations
* **Data Management:** Cleaned and structured raw administrative logs into a verified `.csv` format for seamless reporting.
* **Operational Metric 1:** Identified a bottleneck in resource allocation during peak hours and streamlined team tracking.
* **Operational Metric 2:** Standardized data entry fields to improve data integrity across department logs.
